SHASTHA ENGINEERING WEBSITE - DEPLOYMENT READY PACKAGE

This package was prepared for hosting from the source code.

Best option: Vercel or Netlify using Git import.

Folder containing the website source:
  src/frontend

What was added:
- netlify.toml
- vercel.json
- src/frontend/public/_redirects

Important:
- The original ZIP did not include a built dist/ folder.
- Because dependency download is required, the final production build must be generated by the hosting platform during deployment.

NETLIFY METHOD
1. Upload this project to GitHub.
2. In Netlify: Add new project -> Import from Git.
3. Select the repository.
4. Netlify should use:
   Build command: npm install -g pnpm && pnpm --dir src/frontend install --no-frozen-lockfile && pnpm --dir src/frontend build
   Publish directory: src/frontend/dist

VERCEL METHOD
1. Upload this project to GitHub.
2. In Vercel: Add New -> Project.
3. Import the repository.
4. Vercel will use vercel.json in this package.

CUSTOM DOMAIN
After deploy, connect your domain in Netlify/Vercel dashboard.
